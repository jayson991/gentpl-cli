# gentpl-cli

> A Project To Generate Template On Your Own Way Like `create-react-app` | `@vue/cli` | `@angular/cli`

## Installation

```bash
$ npm i -g gentpl-cli
```

## All Commands

```bash
# look for help
$ gentpl -h
$ gentpl --help

# check gentpl-cli version
$ gentpl -V
$ gentpl --version

# init your own project
$ gentpl init

# list all templates
$ gentpl list
```
