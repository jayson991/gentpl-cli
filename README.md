# gentpl-cli

> A Project To Generate Template On Your Own Way Like `create-react-app` | `@vue/cli` | `@angular/cli`

## Installation

```bash
$ npm i -g gentpl-cli
```

## All Commands

```bash
# look for help
$ gentpl -h

# check gentpl-cli version
$ gentpl -v

# init your own project
$ gentpl new <app-name>

# list all templates
$ gentpl list
```

## All Templates

_react-template_
